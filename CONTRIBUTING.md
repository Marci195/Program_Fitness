# Mitentwickeln

Danke, dass du helfen möchtest.

## So kannst du beitragen

- Fehler melden über GitHub Issues
- Übungen ergänzen
- bessere Anleitungen schreiben
- Videos austauschen
- Design verbessern
- Barrierefreiheit testen

## Änderungen vorschlagen

1. Repository forken.
2. Eigene Branch erstellen:

```bash
git checkout -b feature/meine-aenderung
```

3. Änderungen machen.
4. Commit erstellen:

```bash
git add .
git commit -m "Übung ergänzt"
```

5. Pull Request öffnen.

## Medizinischer Hinweis

Bitte keine medizinischen Heilversprechen einbauen. Die App darf Hinweise geben, ersetzt aber keine ärztliche oder physiotherapeutische Beratung.
